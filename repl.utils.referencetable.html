repl.utils.referencetable module¶

class repl.utils.referencetable.ReferenceTable(initdict=[], ref_by=[], key_f=<function ReferenceTable.<lambda>>, value_f=<function ReferenceTable.<lambda>>, sortkey_f=<function ReferenceTable.<lambda>>)[source]¶

Bases: eudplib.core.eudobj.eudobj.EUDObject

Same as EUDArray with contents =

size(=N), key 1, value 1, key 2, value 2, …, key N, value N

However, these key-value pair are lazily collected.

key_f transforms key before registered. rt = ReferenceTable(key_transformer = EPDConstString) rt.AddPair(“Hello”, 3) # transforms “Hello” to EPD(Db())

value_f works similar to key_f
