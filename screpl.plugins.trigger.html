Trigger plugin

Enables condition checking on TRIG triggers

Log following TRIG triggers:

  1. It contains action: ‘Comment(“screpl-condcheck”)’

  2. No Wait() action.

For all checked triggers, it logs results of conditions for each players P1 through P8. For comparison conditions in the trigger, it additionally logs the exact value for the condition.
