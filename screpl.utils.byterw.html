write_f ( fmt , * args ) [source] ¶
Parse formatted string with python (not in-game)

and write it with EUDVariable or ConstExpr

CAUTION: This does NOT make null end

Available formats:

  • %H: hexadecimal value starts with 0x

  • %D: decimal value

  • %I8d: write_s8

  • %I16d: write_s16

  • %I32d: write_s32

  • %I8u: write_u8

  • %I16u: write_u16

  • %I32u: write_u32

  • %S: string with ptr

  • %E: string with epd

  • %C: single character

Write %% to represent %
